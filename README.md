Adidas Sales Performance Dashboard – Excel Project
Overview
This project presents an interactive sales performance dashboard built using Microsoft Excel. The dashboard analyzes Adidas sales data to generate actionable insights related to revenue, operating profit, units sold, product performance, and regional trends.
The objective of this project was to simulate a real-world business reporting scenario and design a dynamic dashboard that supports data-driven decision-making.
________________________________________
Objectives
•	Analyze sales performance across regions, retailers, and product categories
•	Track key performance indicators (KPIs) such as revenue, operating profit, and units sold
•	Identify top-performing products and high-revenue regions
•	Visualize month-wise and region-wise sales trends
________________________________________
Tools and Techniques Used
•	Microsoft Excel
•	Pivot Tables and Pivot Charts
•	Slicers for interactive filtering
•	Advanced Excel formulas (VLOOKUP/XLOOKUP, IF, SUMIFS)
•	Data cleaning and standardization
•	KPI calculation and dynamic dashboard design
________________________________________
Key Features
•	Interactive dashboard with slicers for dynamic filtering
•	Revenue and profit analysis by region and retailer
•	Product-level performance breakdown
•	Month-on-month sales trend visualization
•	Automated KPI summaries using Pivot-based calculations
________________________________________
Data Preparation
The dataset was cleaned and structured to ensure:
•	Consistent date formatting
•	Standardized region and product category names
•	Accurate aggregation of revenue and profit metrics
•	Removal of duplicates and irrelevant fields
This preprocessing ensured reliable reporting and accurate visual insights.
